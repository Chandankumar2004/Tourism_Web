# **App Name**: Delhi Discoveries

## Core Features:

- Hero Section with Image Collage: Display a full-width hero section on the homepage with a background image collage of iconic Delhi landmarks. Incorporate a dark gradient overlay and bold tagline to capture user attention.
- Tourist Attraction Cards: Implement reusable React Card components to showcase tourist attractions, including an image, title, and brief description.
- Food & Cuisine Showcase: Present Delhi's famous street foods and Old Delhi food culture using image sections with overlay text and smooth fade-in animations.
- Shopping & Markets Listing: Display shopping destinations using React Cards with image hover zoom and brief shopping tips.
- Travel Guide Information Cards: Provide essential travel information using info cards with icons and bullet lists for optimal readability.
- Dynamic Travel Recommendations Tool: AI powered recommendation tool to suggest places to visit, cuisines to explore or experiences to enjoy. This tool will make decision to incorporate time of the year, and other facts it can derive from external APIs. The tool provides personalized recommendations.

## Style Guidelines:

- Primary color: Saffron (#FF9933) to represent heritage and culture.
- Background color: Cream (#FFFDD0) to evoke the color of historical monuments, lightly desaturated.
- Accent color: Deep Red (#A62A2A), analogous to saffron, to highlight calls to action and important sections.
- Headings: 'Playfair' serif font for an elegant, heritage-inspired feel.
- Body: 'PT Sans' sans-serif font for clean, modern readability.
- Use culturally relevant icons to represent categories and information points.
- Subtle hover transitions, fade-in effects, and smooth scrolling for enhanced user experience.
